# ピン配置
PIN1はGPIO4
PIn２はGPIO3につながってます。

## 使うツール
- arduino-cli
- オシロスコープ
- ap

## 開発手順
1. プローブを初期化する
2. コードを書く
3. 書き込む
4. APをつかって、プローブを動かして、テストポイントにプローブを動かす
5. オシロスコープで波形を確認する
6. うまくいってるか確認して、うまくいってなかったらコードを修正する（２に戻る）
